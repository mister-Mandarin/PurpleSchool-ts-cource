Написать на TS реализацию Map в виде класса.
Класс Map с методами: set, delete, get, clear
Хранить данные в buckets, hash которых рассчитывать по какой то логике
Если hash одинаковый, элементы bucket связываются друг с другом

Структура данных бакета пусть будет такая
arr = [
[{ id: 'Тест 1', value: 'значение 1'}],
[{id: 'Тест 2', value: 'значение 2'}, {id: 'Тест 2', value: 'значение 3'}],
[],
[{id: 'Тест 3', value: 'значение 4' }]
];
